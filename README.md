# Portfolio Landing

This is a basic portfolio landing page project.

## About

- Developed by Harsh Pandey.
- This project includes a simple HTML landing page with supporting CSS and JavaScript assets.

## Files

- `index.html` - Main landing page.
- `assets/css/style.css` - Styles for the landing page.
- `assets/js/scripts.js` - JavaScript for interactive behavior.

## Usage

Open `index.html` in a web browser to view the landing page.
